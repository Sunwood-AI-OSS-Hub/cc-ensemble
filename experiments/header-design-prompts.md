# ヘッダー画像のデザインプロンプト

**作成日**: 2025-01-18
**担当**: Agent B (ペイン0.2)

## 提案リスト

### 1. Cyberpunk Terminal Collaboration

サイバーパンク風のターミナルUI。ネオンカラーのコードがペイン間を流れる。

```
A wide banner image (16:5) of a sleek dark terminal interface split
into multiple panes, with glowing neon code streams flowing between
them. Each pane shows a different AI agent working collaboratively -
one writing code, another debugging, a third analyzing data. Cyan and
magenta terminal text, holographic data visualizations floating above
the panes, subtle circuit board patterns in the background.
Minimalist, modern, tech-focused aesthetic. High contrast, clean
lines. --ar 16:5 --style raw
```

**特徴**: ガジェット感が強い、モダン

---

### 2. Abstract Network Playground

抽象的なデジタルアート。浮遊するターミナルウィンドウが光の線で繋がれている。

```
Abstract digital art banner (16:5) showing multiple floating terminal
windows connected by flowing lines of light, representing
collaborative AI agents. The windows contain stylized code blocks and
command prompts. Color palette: deep purples, electric blues, and warm
accent colors. The composition suggests movement and interconnection
between agents. Soft gradients, glassmorphism effects on terminal
windows, subtle particle effects. Modern tech illustration style. --ar
16:5
```

**特徴**: 抽象的、アートっぽい

---

### 3. Minimalist Code Orchestra

ミニマルなデザイン。各ペインが異なる色で光り、接続線でデータが流れる。

```
Clean minimalist banner design (16:5) featuring a row of stylized
terminal panes, each emitting a soft glow in different colors (blue,
purple, green, orange). Between the panes, delicate connection lines
show data flowing between agents. The background is a dark gradient
with subtle geometric patterns. Each pane contains abstract code
representations. Professional, elegant, suitable for a GitHub
repository header. Flat design with subtle depth effects. --ar 16:5
--style raw
```

**特徴**: ミニマル、GitHubに合いそう

---

### 4. Retro-Futuristic CLI Aesthetics

レトロフューチャー風。緑色のCRTモニター美学にネオンアクセント。

```
A retro-futuristic terminal environment banner (16:5) with multiple
tmux-style panes arranged in a grid. Each pane shows a different AI
assistant avatar represented as glowing geometric shapes, with command
outputs scrolling. Green phosphor terminal aesthetic mixed with
modern neon accents. Scanline effects, subtle CRT screen glow at
edges, pixel art elements combined with smooth gradients. Nostalgic
yet contemporary. --ar 16:5
```

**特徴**: レトロフューチャー、懐かしい感じ

---

### 5. Zen Garden of Code

禅風のデザイン。浮遊するターミナルペインが日本庭園の石のように配置され、光の流れで繋がれる。

```
Serene, Zen-inspired banner (16:5) showing multiple floating terminal
panes arranged like stones in a Japanese garden. Soft flowing streams
of light (representing data/code) connect the panes. Muted color
palette with calming blues and greens. Each pane contains harmonious,
clean code snippets. The composition evokes focus and collaborative
flow. Minimalist, elegant, with subtle organic shapes mixed with
technical elements. --ar 16:5 --style raw
```

**特徴**: 禅っぽくて落ち着く雰囲気、優雅
